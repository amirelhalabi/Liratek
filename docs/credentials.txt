## 2. Database (SQLite)
*   **Type:** SQLite 3 (Local File)
*   **Location:** 
    *   **Dev:** `[Project Root]/phone_shop.db` (usually) or inside the Electron User Data folder depending on runtime.
    *   **Prod:** `%APPDATA%/phone_shop/phone_shop.db` (Windows) or `~/Library/Application Support/phone_shop/phone_shop.db` (macOS).
*   **Credentials:** None (File-based access mode).

## 3. Development Accounts (Seeds)
