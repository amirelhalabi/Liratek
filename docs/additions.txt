# Project Manager Additions & Recommendations

## 1. Security & Audit Logs (Critical)
*   **Action Log:** The current schema lacks a comprehensive "User Activity Log". If money goes missing, you need to know who opened the drawer, who edited a stock item, or who changed an exchange rate.
    *   *Recommendation:* Create an `activity_logs` table (user_id, action, timestamp, details_json). Log *everything* related to money or stock.
*   **Granular Permissions:** "Admin vs Staff" is often too binary.
    *   *Recommendation:* specific flags like `can_refund`, `can_view_cost_price`, `can_edit_debt`.
*   **Shift Handovers:** "Blind Closing" is mentioned, but we should enforce a "Blind Count" where the staff enters what they *have* before seeing what the system *expects*.

## 2. Hardware & Integration
*   **Thermal Printer Template:** define the exact width (58mm vs 80mm) and ensure the receipt layout is customizable.
*   **Barcode Scanner Config:** Ensure the app handles scanner input (keyboard wedge mode) correctly without submitting forms prematurely.
*   **UPS (Uninterruptible Power Supply):** Since this is a desktop app with a local DB, a power cut during a write operation could verify corruption (though SQLite is ACID compliance, OS crashes can be messy). Strongly recommend a UPS for the shop PC.

## 3. Financial Robustness
*   **Exchange Rate History:** You store a snapshot in sales, but having a historical graph of the Rate changes helps analyze profit trends vs currency fluctuation.
*   **Drawer Limit Alerts:** Notification if the physical cash in the drawer exceeds a certain limit (security risk), prompting a "Safe Drop".

## 4. User Experience (UX)
*   **Offline Indicator:** A clear visual cue when the app is "Offline" vs "Syncing" vs "Synced".
*   **Keyboard Shortcuts:** Speed is key in a phone shop. Implement global hotkeys:
    *   F1: New Sale
    *   F2: Search Product
    *   F5: Sync Now
    *   Esc: Clear/Cancel
*   **"Quick Cash" Buttons:** In the payment modal, add buttons for common bills (e.g., $10, $20, $50, 1M LBP, 2M LBP) to calculate change instantly.

## 5. Data & Backup
*   **Local Backup Strategy:** Don't rely solely on Cloud Sync. The app should auto-dump a `.sqlite.bak` file to a secondary folder (e.g., an external HDD or Google Drive folder on the PC) on every "Close Shift".

## 6. Business Logic Gaps
*   **Warranty Printing:** The receipt for a sold phone should explicitly print the IMEI and Warranty Terms.
*   **Bundles/Kits:** Logic to sell a "Charger + Cable" bundle for a cheaper price than buying them separately.
